The Auro Version Bot scans every consumer repo in Alaska-ECommerce for outdated Auro design-system packages and files Azure DevOps tickets recommending the upgrade — so individual teams don't have to manually track which repos are behind.

Read-only scenarios describe or preview what the bot would do. Live scenarios — labeled with a red pill — create real ADO tickets and require a confirmation click.
